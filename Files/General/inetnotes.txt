	here are some functions & protocols assign them to one of the 3 planes ( data plane- data and traffic processing, management plane - device info, interact with user etc, control plane - access   https://blog.ipspace.net/2013/08/management-control-and-data-planes-in.html) 
	a) network address Translation (NAT) - data plane
	b) remote access to the command line - control plane
	c) reading the temperature of the router chassis - MP
	d) routing protocol - CP
	e) forward information base - FP ( DP)
	f) the SNMP protocol  - MP
